# QE Reading List
A paper list for machine translation quality estimation.

Welcome to add and criticism.

- **[WMT 2017]** Predictor-Estimator using Multilevel Task Learning with Stack Propagation for Neural Quality Estimation [[paper]](https://aclanthology.org/W17-4763.pdf)
